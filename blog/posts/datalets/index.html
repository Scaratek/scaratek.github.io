Datalets

What is a Datalet?

Datalets are similar to Bookmarklets, however

Instead of injecting JS code it runs HTML code

Also it opens a about:blank with that code instead of directly injecting it

Datalets can be used like data:text/html,

Then using script tags we can execute JS code

So far what I've managed to make using Datalets are

- Forkbomb (funny)

- History Flooder

- Tool for Bypassing Securly for Classrooms

The Bypass for Securly (aka Cekzu) works by exploiting the fact that Securly cannout access about:blank tabs

It works by have the user input a site and iframe styled to be fullscreened

Securly cannot view about:blank tabs so you can access a site without the teacher seeing

However sites are still blocked as Securly uses webRequests for blocking sites

Securly cannot access about:blank tabs as they are treated as chrome:// pages which extensions cant access due to security reasons
