Pollen

Pollen is a User Policy editor for CrOS

It allows users with Devmode enabled on their Chromebook

To modify their schools User Policy set in GAC

This can be useful for bypssing restrictions set by the school

Without needing to Unenroll

Pollen works by overriding a JSON file

That stores your schools User Policy

By default it overrides the file to have no restrictions

however you can custom this to your liking

Pollen does not work apon restart, unless you have RootFS disabled

If you do have RootFS enabled Pollen will simply set the default policy apon restart
