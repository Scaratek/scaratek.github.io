FlightOS

FlightOS is a fork of Arch Linux that is designed to have a Trans Gender theme.

It is designed to have a sleek and modern UI like MacOS

It also has its own Desktop Environment called LucidDE.

LucidDE is a configured version of Plasma 5

It also has its own Package Manager called Unicorn.
