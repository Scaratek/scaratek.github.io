Cekzu

Cekzu is a script that Bypasses Securly Classroom

It works by inserting a iframe into a about:blank tab using data:text/html,

data:text/html, allows you to directly write HTML code into a blank tab

Then the iframes source can be controlled by prompt

It also has a tab spoofing feature to change the tab title

It bypasses Securly Classroom as Securly cannot modify pages like about:blank and data:text/html,
