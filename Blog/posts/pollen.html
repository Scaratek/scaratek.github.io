Pollen

Pollen is a Bash Script that allows users to modify their schools Chromebook User Policies

Chrome OS has a undocumented feature that allows you to create a JSON file with policies

Once you have the JSON file you can write the file to a directory in CrOS that over writes the current policies

It works similar to Crome policies on Windows

With this knowledge we can make a Bash Script

That creates a JSON file and overwrites the current one

Although we need to use a special method to apply the JSON file

As you cannot overwrite the file directly as it is protected by RootFS

The side effects of this is that you need to apply Pollen every time you boot the Chromebook
