Datalet

A Datalet is similar to a Bookmarklet, execpt its HTML code instead of JS code

To use a Datalet in the URL bar paste in "data:text/html,"

After the , you can start writing HTML code

So you might think, using a script tag can't we just get Bookmarklets back?

Sadly not, data:text/html, does not write to the current tab

Instead it creates a new tab it writes the HTML code to that tab

The tab functions similar to a about:blank tab

These tabs act similar to chrome:// pages

Which means no extension can modify it, including Securly
